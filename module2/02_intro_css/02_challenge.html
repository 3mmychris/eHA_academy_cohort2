Computer Programming: Intro to HTML/CSS

The purpose of this course is to teach you how to use HTML and CSS to make webpages.
Hypertext Markup Language (HTML) is the standard markup language for pages designed
to be displayed in a web browser. Cascading Style Sheet (CSS) is the stylesheet language that
describes the style of an HTML page, such as change the color, font, layout, and more.This
course is part of a free 2 month web development training organised by eHealth Africa that
will help learners learn and apply the necessary coding skills needed to start a career in web
development. Learners will attend a bi-weekly in-person workshop at the eHealth Africa Kano
Campus and complete coursework online during the week.Course ObjectivesLearn how to use
